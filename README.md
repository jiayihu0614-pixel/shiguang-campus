# 拾光校园 - 校园活动发现平台

## 项目主题

面向大学生的校园活动发现、筛选、收藏、报名与日程管理网站。主题积极、内容原创，适合作为《Web应用开发》期末作品。

## 运行方法

直接双击 `index.html`，使用 Chrome、Edge 或 Firefox 打开即可。项目不依赖服务器、数据库或外部 CDN，断网也能完整演示。

## 主要功能

1. 学生登录：首次使用需输入姓名、学号、年级和班级，避免固定用户信息。
2. 首页：沉浸式校园视觉、精选活动、兴趣分类、校园故事。
3. 活动广场：关键词搜索、分类筛选、时间筛选、热度排序和空状态。
4. 活动详情：活动介绍、时间、地点、主办方、名额与报名状态。
5. 收藏功能：支持收藏和取消收藏，数据保存在浏览器 `localStorage`。
6. 报名功能：表单校验、报名状态更新、自动加入个人日程。
7. 我的日程：日历标记、活动列表、取消报名。
8. 个人中心：展示学号、姓名、年级、班级、参与数据、收藏列表、动态参与足迹、兴趣标签与成就进度。
9. 响应式设计：适配电脑、平板和手机。
10. 深色模式：支持主题切换并记住用户选择。
11. 可访问性：语义化标签、键盘关闭弹窗、焦点跳转和图片替代文字。

## 技术说明

- HTML5：语义化页面结构、表单与可访问性属性。
- CSS3：Grid、Flexbox、媒体查询、CSS 变量、动画和深色主题。
- JavaScript：学生登录、数据渲染、路由切换、筛选排序、弹窗、表单校验和本地存储。
- 工程结构：HTML、CSS、JavaScript 和图片资源分离，便于维护。

## 建议答辩演示流程

1. 介绍主题、用户群体和信息架构。
2. 首次进入页面，输入姓名、学号、年级和班级完成学生登录。
3. 展示首页及移动端响应式布局。
4. 在活动广场搜索“摄影”，再按分类和热度筛选。
5. 打开活动详情，收藏并提交报名。
6. 进入“我的日程”，展示报名数据自动同步。
7. 在个人中心查看学生资料和收藏，再演示深色模式和刷新后的数据保留。

## 可继续扩展

接入 Vue 或后端 API 后，可进一步实现用户登录、活动发布、管理员审核、真实数据库和消息提醒。
