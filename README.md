# 拾光校园 - 校园活动发现平台

## 项目主题

面向大学生的校园活动发现、筛选、收藏、报名与日程管理网站。

## 运行方法

直接双击 `index.html`，使用 Chrome、Edge 或 Firefox 打开即可。项目不依赖服务器、数据库或外部 CDN，断网也能完整演示。

## 主要功能

1. 首页：沉浸式校园视觉、精选活动、兴趣分类、校园故事。
2. 活动广场：关键词搜索、分类筛选、时间筛选、热度排序和空状态。
3. 活动详情：活动介绍、时间、地点、主办方、名额与报名状态。
4. 收藏功能：支持收藏和取消收藏，数据保存在浏览器 `localStorage`。
5. 报名功能：表单校验、报名状态更新、自动加入个人日程。
6. 我的日程：日历标记、活动列表、取消报名。
7. 个人中心：参与数据、收藏列表、兴趣标签与成就进度。
8. 响应式设计：适配电脑、平板和手机。
9. 深色模式：支持主题切换并记住用户选择。
10. 可访问性：语义化标签、键盘关闭弹窗、焦点跳转和图片替代文字。

## 技术说明

- HTML5：语义化页面结构、表单与可访问性属性。
- CSS3：Grid、Flexbox、媒体查询、CSS 变量、动画和深色主题。
- JavaScript：数据渲染、路由切换、筛选排序、弹窗、表单校验和本地存储。
- 工程结构：HTML、CSS、JavaScript 和图片资源分离，便于维护。
